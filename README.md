# F1 Mercado

App estatica de mercado de pujas para liga F1 Manager.

## Publicacion

Este repo esta preparado para GitHub Pages desde la rama `main`, carpeta `/`.

Archivos publicos:

- `index.html`
- `app.js`
- `styles.css`
- `save-reader.js`
- `firebase-config.js`

La seguridad de datos depende de Firebase Authentication y Realtime Database Rules.
